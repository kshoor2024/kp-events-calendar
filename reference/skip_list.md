## How Cowork handles this list

1. If an event on this list appears in a source scan: add to database with appropriate "Do Not Outreach" tag.
2. Business events: tag = "Do Not Outreach — Business", Type = "Business", no outreach draft generated.
3. International events: tag = "Do Not Outreach — International", Type = usually Business, no outreach draft generated.
4. Unverified/dead events (High Times cups etc.): tag = "VERIFY — may not be happening", do not add to outreach queue until Brandon confirms.
5. Any satellite event of a skip-listed parent that IS a fit (e.g., a party the night before MJBizCon that IS consumption-friendly) — add as its own database entry. The skip entry doesn't cascade to adjacent events.
